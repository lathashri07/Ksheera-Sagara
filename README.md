# Ksheera-Sagara Hub 🐄🥛

**Empowering Dairy Farmers with Data-Driven Insights**

Ksheera-Sagara Hub is a specialized mobile application designed for dairy farmers to digitize farm management. It moves traditional paper-based logging into a secure, real-time digital environment, allowing farmers to track the profitability of individual cows and the overall financial health of their farm.

## 🌟 Key Features

- **Secure Farmer Login:** Validated access using phone numbers (10 digits) or email.
- **Individual Cow Tracking:** Detailed logs for up to 10 cows, monitoring milk volume and fat percentage.
- **Expense Management:** Categorized logging for Fodder, Medical, and Labor costs.
- **Visual Analytics:** Interactive Pie Charts for expense breakdown and high-profit cow identification.
- **Financial Health Status:** Real-time calculation of net savings with a color-coded health dashboard.
- **PDF Reporting:** Generate professional monthly financial summaries with transaction dates for sharing or printing.

## 🛠️ Technologies Used

- **Language:** Kotlin
- **UI Framework:** Jetpack Compose (Material 3)
- **Backend:** Firebase Realtime Database
- **Architecture:** MVVM (Model-View-ViewModel)
- **Theme:** Nature-inspired Green UI for enhanced accessibility for rural users.
- **PDF Engine:** Android Printing Framework
